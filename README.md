# Jaipur.com

Premium luxury Jaipur city media website.

## Stack
React • Vite • Tailwind CSS • react-router-dom • lucide-react

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Netlify
1. Push this folder to a GitHub repo
2. New site from Git → select repo
3. Netlify auto-detects (build: `npm run build`, publish: `dist`)
4. Done — `netlify.toml` already handles SPA routing

Or drag & drop the `dist/` folder into Netlify after `npm run build`.
